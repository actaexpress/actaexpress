SEO ACTAEXPRESS — COUVERTURE NATIONALE COMPLÈTE
===============================================
Date : 13 juin 2026

CHIFFRES
--------
- 34 969 communes (source officielle geo.api.gouv.fr)
- 101 départements (métropole + DOM)
- 8 types de documents
- ~280 000 pages SEO générées
- 7 sitemaps (index : sitemap.xml)

DONNÉES
-------
content/seo/communes-fr.json      — liste compacte (commitée)
content/seo/departements-fr.json  — 101 départements
content/seo/regions-fr.json       — 18 régions

RÉGÉNÉRER
---------
  cd scripts
  python fetch-communes-fr.py    # met à jour les JSON (1× par an)
  python generate-seo-geo.py     # ~2 min, ~280 000 fichiers

DÉPLOIEMENT NETLIFY
-------------------
netlify.toml lance generate-seo-geo.py au build.
Le dossier geo/ est dans .gitignore (trop volumineux pour Git).

Option A — Build Netlify (recommandé) :
  git push → Netlify génère les pages au déploiement

Option B — Build local puis deploy :
  python scripts/generate-seo-geo.py
  netlify deploy --prod --dir .
